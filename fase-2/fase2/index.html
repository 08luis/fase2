Flex Box

El módulo de diseño de caja flexible, más comúnmente conocido como flexbox , es una herramienta poderosa para crear diseños con CSS. No es tan poderoso como CSS Grid, que veremos en el Capítulo 12 , pero puede resolver muchos problemas de diseño. Flexbox es un diseño unidimensional que puede colocar elementos horizontal o verticalmente (pero no ambos). Un elemento que usa flexbox como diseño se conoce como contenedor flexible , y los elementos dentro de él son elementos flexibles .

Attardi, J. (2020). Flexbox. En Modern CSS (págs. 205-228). Apress, Berkeley, CA.
